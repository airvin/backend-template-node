# Backend Template with Node.js and Express

## Dependencies

- [nodemon](https://nodemon.io/)


## Project structure
```
.
├── app.js
├── bin
│   └── www
├── package.json
├── public
│   ├── images
│   ├── javascripts
│   └── stylesheets
│       └── style.css
├── routes
│   ├── index.js
│   └── users.js
```

## Setup directory for a new project

```
git clone git@github.com:airvin/backend-template-node.git <project-name>
cd <project-name>
rm -rf .git
```

Install dependencies

```
npm install
```

## Running the server

```
npm start
```
